### About

I'm a full-stack developer based in India. I like problems where the data is messy, the latency matters, and the UI has to make sense at 9:15 AM with money on the line.

Most of my time goes into building **real-time, data-heavy systems** — currently a personal trading analytics platform that ingests Nifty 50 options data from the Zerodha Kite WebSocket, stores it in TimescaleDB, and surfaces dealer-positioning signals (GEX, Vanna, Charm) through a React dashboard.

I work the full stack — schema design, ingestion pipelines, signal logic, API layer, frontend — and I prefer shipping small, working things over speculating in docs.

### What I'm building right now

- **Kite Options Analytics** *(private)* — TimescaleDB schema for raw tick collection (option / futures / spot / VIX), forward-blind signal engine using OI deltas, volume absorption, futures OI classification. React + Recharts frontend with GEX page, Dealer Pressure, Bubble Map, FII/DII flow, GEX Regime views.

### Private Projects

- **ERP-based Logistics Management System** — A monolithic ERP system for logistics operations, built with Python and PostgreSQL. In the process of migrating to a microservices architecture for better scalability and maintainability.
